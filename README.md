# MPJ Event & Ticketing System

Sistem manajemen event modern yang dibangun dengan Next.js 14, dirancang untuk mempermudah pendaftaran peserta, manajemen tiket, dan absensi berbasis QR Code.

## 🚀 Fitur Utama
- **Listing Event**: Antarmuka responsif untuk melihat event mendatang.
- **Registrasi Dual-Path**: Mendukung jalur NIAM (Anggota) dan Umum.
- **Ticketing & QR**: Pembuatan tiket otomatis dengan QR Code unik.
- **QR Scanner**: Modul absensi panitia berbasis web (Mobile-First).
- **Admin Panel**: Kelola event, peserta, dan narasumber dalam satu tempat.

## 🛠️ Tech Stack
- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS.
- **UI**: shadcn/ui, Lucide Icons.
- **Utilities**: Axios, html5-qrcode, qrcode.react.

## 📦 Instalasi
```bash
npm install
```

## 💻 Pengembangan
```bash
npm run dev
```
